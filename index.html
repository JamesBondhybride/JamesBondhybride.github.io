About My Programming Languages

How i started coding, October 13, 2023

I first started programming at Fontys University of Applied Science with C# programming, following online tutorials and courses that taught me the basics of various languages and frameworks. I started with HTML, CSS, and JavaScript, which are the building blocks of web development. I learned how to create static and dynamic web pages, add interactivity and functionality, and use tools like Bootstrap and jQuery to enhance the user experience. I also learned how to use Git and GitHub to manage my code and collaborate with other developers.
After mastering the web development skills, I wanted to learn more about data science and machine learning. I enrolled in a course on Python, which is a popular language for data analysis and manipulation. I learned how to use libraries like NumPy, Pandas, Matplotlib, and Scikit-learn to perform various tasks such as data cleaning, visualization, modeling, and evaluation. I also learned how to use Jupyter Notebook, which is an interactive environment for writing and running code. As I progressed in my learning journey, I became interested in exploring other domains and technologies. I learned how to use C#, which is a general-purpose language that can be used for desktop, web, mobile, and game development. I learned how to use Visual Studio Code, which is a powerful code editor that supports multiple languages and extensions. I also learned how to use Unity, which is a cross-platform game engine that allows me to create 2D and 3D games for various platforms.
I learned programming by being curious, motivated, and persistent. I enjoyed the process of solving problems, creating projects, and learning new things. I also benefited from the online community of programmers who shared their knowledge, feedback, and resources. Programming has become my passion and hobby, and I hope to continue learning and improving my skills.
